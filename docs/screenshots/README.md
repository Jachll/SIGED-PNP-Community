# Screenshots seguros

Este directorio esta reservado para capturas publicas generadas con datos sinteticos.

Antes de agregar una imagen:

- confirma que no muestre datos policiales reales;
- confirma que no muestre mapas institucionales privados;
- confirma que no exponga rutas internas, usuarios reales, tokens, credenciales o URLs privadas;
- usa solamente datos de `database/sample_data/` u otros datos claramente sinteticos;
- nombra los archivos con prefijo `synthetic-`, por ejemplo `synthetic-dashboard.png`.

Si hay duda sobre el origen de una captura, no la agregues al repositorio publico.
