# Datos de ejemplo

Los archivos de esta carpeta son sintéticos.

- No contienen datos reales.
- No contienen información policial real.
- No contienen ubicaciones operativas reales.
- Solo sirven para pruebas locales, validación del ETL y demostraciones.

Si necesitas usar datos institucionales, mantenlos fuera del repositorio público. Usa rutas locales ignoradas por Git y valida permisos, anonimización y clasificación de la información antes de compartir cualquier archivo.
